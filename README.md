## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura básica do projeto.
- **CSS3**: Estilização dos elementos da página.
- **JavaScript**: Lógica de animação utilizando Anime.js.
- **Anime.js**: Biblioteca para animações JavaScript.
